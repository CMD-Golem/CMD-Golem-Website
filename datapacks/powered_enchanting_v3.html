Powered Enchanting

1.15 +

This Data Pack changes the entire enchanting system of Minecraft. In addition, there are a bunch of add-ons with more custom Enchantments, which you can put together according to your needs.

In total there are three different Data Pack Editions.

  • The CMD-Golem Edition, which adds 17 new Enchantments.
  • The Vanilla Edition, which only contains the Vanilla Minecraft Enchantments.
  • And the Empty Edition, which doesn't contain any Enchantments.

To each Edition you can install Addons according to your needs.

Build the Enchanting Table

First craft the normal Enchanting Table and place it. Then you need to build the pillars around it. To simplify this process, you can simply drop a Glowstone Dust on the Enchantment Table. At the four places where blue particles can be seen, you need to place two Lapis Lazuli Blocks on top of each other, and at the four white particles, you need to place Chiseled Quartz Blocks.

To start the enchantment process, you need to make Lapis Lazuli Slices. For each enchantment you create, one of them will be consumed.

Enchanting

To start enchanting, you need to drop a book on the Enchanting Table. After that, four books appear near the Enchanting Table and move towards the Quartz Blocks. After they reach the blocks, each book gets an enchantment, which is displayed as a floating text above the book. Now you can choose one of the enchantments by placing the Lapis Lazuli Slice on the Quartz Block below it.

Enchanting books requires 30 experience levels, two of which are consumed.

Advanced Enchanting
To get rarer enchantments like Mending, you need to replace the four Quartz Blocks with Obsidian and place an End Crystal on each of them. Everything else works the same as normal enchanting.
Advanced enchanting of books requires 50 experience levels, eight of which are consumed. Also, the End Crystal placed on the selected pillar will explode.

Combining

To get an enchantment on a tool or armor, you need to drop it on an Enchanting Table along with the Enchanted Book. You can also combine the enchantments of two different items. If you do that, the first item you dropped will be preserved and gets the enchantments of the second one.

Each time you enchant the item, the experience cost of it increases.
The formula for the calculation is: ( times of enchanting + 1 )2
Warning: The combination in the Anvil will no longer work.
