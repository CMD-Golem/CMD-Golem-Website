NBt-Crafting

Create a recipe for the Crafting Table that has an item with NBT data as result.

  1. First define the name of the Data Pack ID and select the Minecraft version you want to use for the Data Pack.
  2. Go to the first dark gray section, which contains all text fields for the first recipe. Paste the JSON code for the recipe (generator) and the result item with NBT in the according inputs.
    • You can add a new Recipe by clicking on the according button in the navigation bar or by using the key combination Ctrl + Enter.
    • Navigate quickly between all recipes by using the buttons in the left sidebar.
    • Each file path can be changed in the spoiler "Advanced options". The grey placeholder text is the current path and will be used unless you overwrite it.
  3. To download the finished Data Pack you can again use the button in the navigation bar or the key combination Ctrl + S.

Create your Recipe

The ingridients of your defined JSON recipe are used to craft the item with NBT. Also, the count of your result item will be adopted.

The only thing to note is that the result item, defined by the JSON recipe, will show up in the crafting table. But as soon as you take the item in your Inventory, it will be cleared and the item with NBT will be given to you.

Since the clear command clears the JSON result item, it can't be an item that could be in your inventory with NBT data on it. I recommend using items like Barriers, Structure Voids or Knowledge Books (for non-stackable items) instead.

Automate file path generation for new recipes

To make it easier to create multiple recipes, I have given the $ character a special use.

If you put it as first letter in File Name fields, the name will be copied into new recipes. Remember to adjust the folder path in order to prevent overlaps.

If you use the $ character in Folder Path fields, it will be replaced with the Recipe Name.

To define the Recipe Name, you have to activate the checkbox in the spoiler "Advanced Options" bellow the version selection. Afterwards, each recipe receives a new input field in which the recipe name can be entered.

Since a normal Recipe Unlocker Data Pack would break this crafting method, you should use this one.
