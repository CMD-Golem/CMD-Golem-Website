CMD-Golem

Installation

Data Packs

  1. Go to the Singelplayer World selection and select the world where you want the Data Pack in.
  2. Click on Edit at the bottom.
  3. Click on open World Folder.
  4. Open the folder datapacks and paste the .zip File of the Data Pack into it.

Resource Packs

  1. Open the Options Menu
  2. Click on Resource Packs
  3. Click on Open Pack Folder
  4. Paste the .zip File of the Datapack into the folder.

Maps

  1. Open the Options Menu
  2. Click on Resource Packs
  3. Click on Open Pack Folder
  4. Go to the parent directory
  5. Open the folder saves
  6. Paste the unzipped File of the Map into the folder.
