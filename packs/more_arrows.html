More Arrows

1.14 +

This Data Pack adds a lot new Arrows. Like TNT Arrow, Teleport Arrow and so on.

It also changes the way to craft Tipped Arrows. Now the crafting for them works like the crafting for the new Arrows. You need to drop a normal Potion or a Splash Potion on a Fletching Table and shoot it with a Bow.

Crafting

Drop the matching item, which you can see in the table underneath in the Arrow spoiler, on a Fletching Table and shoot the item with a Bow or Crossbow. (You can't use an Infinity Bow and you need to be in survival or adventure mode)

If you have OP or cheats enabled you can get all Arrows with the following command:

/function more_arrows:give

Settings

Disable crafting Tipped Arrows with normal Potions.

/scoreboard players set #potion more_arrows 1

Disable crafting Tipped Arrows with Splash Potions.

/scoreboard players set #splash_potion more_arrows 1

Disable Arrows. Replace #id with the id of the according Arrow from the table bellow.

/scoreboard players set #id more_arrows 1

To enable the Setting again you need to execute the same Command except of the 1 replace with a 0.
